Создай одностраничное HTML-приложение Todolinks — локальный менеджер ссылок и задач с единым списком. Хранилище данных — IndexedDB.

=== 1. ОБЩИЕ ТРЕБОВАНИЯ ===
- Весь код в одном файле index.html (HTML, CSS, JS).
- Открывается двойным кликом, без сервера, Node.js, Docker.
- Хранилище: IndexedDB (база данных TodolinksDB, версия 2).
- Чистые HTML/CSS/JS, без фреймворков и CDN.
- Две темы: светлая / тёмная (переключатель — одна круглая кнопка ☀/☾).
- Адаптивный дизайн (ПК, планшет, мобильные).
- Название вкладки: «Todolinks · ссылки и задачи».
- Верхняя панель и статистика закреплены, прокручивается только список.
- При фокусе на поиске — мягкое изменение фона на `var(--accent-muted)` и границы на `var(--accent)`, без обводки (outline: none).
- При загрузке страницы — автоматический фокус на поле поиска (через setTimeout 50ms).
- Подсказки под поисковой строкой **отключены** (не отображаются).

=== 2. СТРУКТУРА ДАННЫХ (IndexedDB) ===
База данных: TodolinksDB, версия 2.
Хранилище объектов: items (ключ: id).
Индексы: type, todoStatus, dueDate, pinned, createdAt.

Запись (ссылка):
{
  id: "uuid", type: "link", title: "GitHub", url: "https://github.com",
  tags: ["работа"], pinned: false, description: "", color: null,
  createdAt: "ISO8601"
}

Запись (задача):
{
  id: "uuid", type: "todo", title: "Купить молоко", description: "",
  tags: ["дом"], pinned: false, todoStatus: "active", dueDate: "2026-05-01",
  completedAt: null, repeat: null, color: null,
  createdAt: "ISO8601"
}

Поля:
- repeat: null | "daily" | "weekly" | "monthly" | "yearly" — повтор задачи
- color: null | "1"..."9" — цвет фона строки

=== 3. ВЕРХНЯЯ ПАНЕЛЬ ===
Слева направо:
1. Поисковая строка (растягивается). Placeholder: «Введите запрос, ссылку или задачу...»
2. Кнопка + (круглая, синяя) — открывает форму новой ссылки.
3. Кнопка Экспорт ↑ (круглая, зелёная).
4. Кнопка Импорт ↓ (круглая, красная).
5. Кнопка темы ☀/☾ (круглая, серая).

Все круглые кнопки: 40x40px, border-radius 50%, display:flex, align-items:center, justify-content:center, font-size:1.4rem, font-weight:bold, border:none, color:white, transition:opacity 0.1s.
Цвета: + (--accent), Экспорт (--export-bg), Импорт (--import-bg), тема (--button-bg).

=== 4. ФИКСИРОВАННАЯ ШАПКА ===
.app: max-height: 100vh, overflow: hidden, flex-колонка.
.top-bar, .stats-bar: flex-shrink: 0.
.list-container: overflow-y: auto, flex: 1.

=== 5. СТАТИСТИКА ===
Три равные колонки (flex с равными flex: 1):
- Левая: ссылки «GitHub» и «♥ Поддержать» (внешние, открываются в новой вкладке).
- Центр: «Всего: X ссылок, Y задач (Z выполнено)».
- Правая: кнопка «ℹ Помощь» (синий оттенок) и кнопка «🗑 Удалить всё» (красноватый оттенок).

Ссылки и кнопки имеют полупрозрачность, при наведении становятся ярче.

=== 6. УМНАЯ СТРОКА ===
- Live-поиск по title, url, description, tags.
- Поиск выполненных задач: «done», «готово», «выполнено», «completed», «finished» и т.п. (если одно из слов есть в запросе, показываются только завершённые задачи).
- Автоопределение типа:
  - URL (начинается с http://, https:// или содержит домен вида example.com) → ссылка.
  - Содержит дату YYYY-MM-DD или @тег → задача.
  - Обычный текст → открытие формы задачи с предзаполненным названием.

=== 7. БЫСТРЫЙ ПОИСК НА САЙТАХ ===
При вводе ключевого слова и поискового запроса через пробел — переход на сайт с поиском.

Ключевики и URL:
- яндекс, yandex → https://yandex.ru/search/?text=
- гугл, google → https://www.google.com/search?q=
- ютуб, youtube → https://www.youtube.com/results?search_query=
- рутуб, rutube → https://rutube.ru/search/?q=
- озон, ozon → https://www.ozon.ru/search/?text=
- валберис, wildberries, вб, wb → https://www.wildberries.ru/catalog/0/search.aspx?search=
- маркет, market → https://market.yandex.ru/search?text=
- вики, wiki → https://ru.wikipedia.org/w/index.php?search=
- гитхаб, github → https://github.com/search?q=
- переводчик, перевод, translate → https://translate.yandex.ru/?text=
- нейросеть, нейро, perplexity, перплексити → https://www.perplexity.ai/search?q=

=== 8. ПОВЕДЕНИЕ ПО ENTER (приоритет) ===
1. Проверка на поисковик/переводчик → открыть поиск/перевод в новой вкладке.
2. Активный элемент (keyboard-active) → открыть ссылку в текущей вкладке / переключить задачу (с учётом повтора).
3. Пустой ввод → открыть форму новой ссылки.
4. Есть результаты поиска внутри Todolinks → ничего не делать.
5. URL (распознанный по http://, https:// или домену) → открыть форму ссылки с предзаполненным URL.
6. Задача (содержит дату YYYY-MM-DD или @тег) → быстрое создание без модалки.
7. Обычный текст → открыть форму новой задачи с предзаполненным названием.

=== 9. НАВИГАЦИЯ С КЛАВИАТУРЫ ===
- ↓/↑ — подсветка строк (класс .keyboard-active, фон --active-row).
- Enter на ссылке — открыть URL в текущей вкладке.
- Enter на задаче — переключить чекбокс выполнения (с учётом повтора).
- Shift+Enter — редактирование.
- Esc — снять выделение, очистить поиск.

=== 10. ЕДИНЫЙ СПИСОК ===
Сортировка (плоская, без вложенности):
1. Закреплённые (pinned) в начале своей группы.
2. Активные задачи со сроком (по dueDate, ранние сверху).
3. Активные задачи без срока.
4. Ссылки.
5. Выполненные задачи (в конце).

Строка (всё в одну линию, gap 0.8rem):
- Чекбокс выполнения (только задачи, accent-color: --accent, filter grayscale(0.3) opacity 0.8).
- Заголовок: для ссылок — кликабельный текст с атрибутом title, показывающим URL; для задач — обычный текст (выполненные — зачёркнуты).
- Срок (для задач). Если просрочен — класс .overdue.
- Описание (max-width: 400px, text-overflow: ellipsis).
- Теги (справа). Отображаются **всеми заглавными буквами** (CSS: text-transform: uppercase).
- Чекбокс массового выбора (правый).
- Кнопка закрепа: ⚐ (белый флаг) / ⚑ (красный флаг). Кнопка фиксированной ширины 34px.
- Кнопка ✎ (редактировать, цвет --text-secondary).
- Кнопка ✕ (удалить, цвет --delete-color).

**Важно:** URL ссылки в строке не отображается (только в title при наведении и в форме редактирования).

Двойной клик по строке открывает редактирование элемента.

Повторяющиеся задачи:
- При отметке чекбокса (выполнении) задача не уходит в архив, а её dueDate обновляется по правилу (используется UTC для надёжности):
  - daily: +1 день (setUTCDate)
  - weekly: +7 дней
  - monthly: +1 месяц с сохранением числа месяца. Вычисляется через Date.UTC для предотвращения сдвигов часовых поясов.
  - yearly: +1 год
- Статус задачи остаётся active, completedAt не устанавливается.
- При снятии чекбокса задача переходит в активный статус.

Цвета строк:
- Выбор в форме редактирования из списка:
  По умолчанию, Мятный, Розоватый, Голубоватый, Небесный, Персиковый, Сиреневый, Дымчатый, Лимонный, Румяный
- Применяются как дополнительный фон строки (класс .color-1 … .color-9).

=== 11. МАССОВЫЕ ОПЕРАЦИИ ===
При выборе чекбоксов (через Shift можно выделить диапазон) появляется нижняя панель .bulk-bar с удалением и снятием выделения.

=== 12. ФОРМА ДОБАВЛЕНИЯ / РЕДАКТИРОВАНИЯ ===
Переключатель Ссылка / Задача.
Поля:
- Название *
- URL * (ссылка). При вводе автоматически добавляется https://, если протокол не указан.
- Срок (задача)
- Повтор (задача): Без повтора / Ежедневно / Еженедельно / Ежемесячно / Ежегодно
- Цвет (селект с названиями).
- Теги (через запятую). При вводе показываются подсказки (.tag-suggestions) с уже существующими тегами из базы. При клике на подсказку тег добавляется. Вводимые и отображаемые теги приводятся к верхнему регистру.
- Описание
Кнопки: Отмена / Сохранить.

=== 13. ИМПОРТ / ЭКСПОРТ ===
- Экспорт (↑):
  - Используется File System Access API (showSaveFilePicker) с предложенным именем Todolinks_YYYYMMDD_HHMMSS.json.
  - **Важно:** Если API не поддерживается (например, в Safari), реализован fallback — стандартное скачивание через `<a download>`.
- Импорт (↓): модалка «Заменить все» / «Добавить».

=== 14. МОДАЛЬНЫЕ ОКНА ===
- Единый стиль: белый/тёмный фон, скруглённые углы, мягкая тень.
- Подтверждение удаления всего и удаления одной записи.
- Окно «Помощь» с кратким руководством: умная строка, быстрые поисковые команды (таблица с ключевиками), горячие клавиши, хранение данных.

=== 15. ДОПОЛНИТЕЛЬНО ===
- Toast-уведомления (3.5с).
- Пустое состояние: «Нет ссылок или задач. Добавьте первую.»
- Автофокус при загрузке: setTimeout(() => smartInput.focus(), 50).
- Внешние ссылки (GitHub, Поддержать) открываются в новой вкладке.
