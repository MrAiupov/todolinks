# 🔗 Todolinks

**Локальный менеджер ссылок и задач в одном HTML-файле.**  
Открывается двойным кликом, работает полностью офлайн, без сервера и интернета.  
Все данные сохраняются в edDB вашего браузера.

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

---

## ✨ Возможности

- **Единый список** — ссылки и задачи вместе, с умной сортировкой  
- **Умная строка** — вводите URL, задачу с датой или тегом, система сама определит тип  
- **Быстрое добавление** — `Enter` создаёт элемент без лишних окон  
- **Навигация с клавиатуры** — стрелки, `Enter`, `Shift+Enter`, `Esc`  
- **Закрепление** — важные элементы всегда сверху  
- **Повторяющиеся задачи** — ежедневные, еженедельные, ежемесячные, ежегодные  
- **Цветовое выделение** — 9 мягких оттенков для визуальной организации  
- **Теги** — быстрый поиск и группировка, автодополнение из существующих  
- **Массовые операции** — выделение через `Shift`, удаление групп  
- **Импорт/экспорт** — сохранение и восстановление данных в JSON  
- **Тёмная и светлая тема** — переключение в один клик  
- **Адаптивный дизайн** — работает на ПК, планшетах и телефонах  

---

## 🚀 Как использовать

1. **Скачайте** файл `Todolinks.html` из [последнего релиза](https://github.com/MrAiupov/todolinks) или сохраните код локально.  
2. **Откройте** его двойным кликом в любом современном браузере.  
3. **Начинайте** добавлять ссылки и задачи прямо с клавиатуры.

---

## ⌨️ Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| `↑` / `↓` | Перемещение по списку (подсветка) |
| `Enter` | Открыть ссылку / переключить задачу |
| `Shift` + `Enter` | Редактировать выбранный элемент |
| `Esc` | Снять выделение, очистить поиск |
| `Shift` + клик | Массовое выделение чекбоксами |

## 💡 Быстрый ввод

| Что вводите | Результат |
|-------------|-----------|
| `https://github.com` | Создаст ссылку (откроется форма) |
| `github.com` | Создаст ссылку (авто‑добавление `https://`) |
| `2026-05-01 Купить хлеб @дом` | Создаст задачу со сроком и тегом `@дом` |
| `Позвонить маме` | Откроет форму новой задачи с названием |

## 💾 Хранение данных

Все данные сохраняются в **edDB** браузера (база `TodolinksDB`).  
Никакие данные не отправляются на сервер — всё остаётся на вашем компьютере.  
Вы можете хранить десятки тысяч записей без потери производительности.

---

## ⬆️ Экспорт и импорт

- **Экспорт (↑)** — сохраняет все данные в JSON‑файл.  
  В современных браузерах открывается окно выбора папки.  
- **Импорт (↓)** — загружает данные из JSON. Можно **заменить** текущие данные или **добавить** к ним.
